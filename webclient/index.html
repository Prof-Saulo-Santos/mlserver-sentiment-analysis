info Sobre este projeto

Este é um classificador de sentimentos servido pelo MLServer utilizando um pipeline do Scikit-Learn (TfidfVectorizer + LogisticRegression). O modelo foi treinado com textos em português e classifica em três categorias: positivo, negativo e neutro.
